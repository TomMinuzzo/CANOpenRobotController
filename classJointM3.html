double JointM3::driveUnitToJointPosition ( int  driveValue)
privatevirtual

Converts from the drive value to the equivalent value for the joint position.

Notes:

  • The drive value is always an integer (due to the CANOpen specification) and the joint value is always a double (data type of position)
  • This may be a linear relationship (e.g. degrees to encoder counts) or a more complicated one (e.g. linear actuator position to degrees) depending on the structure of the device and system.
Parameters
driveValueThe drive value to be converted
Returns
The equivalent joint value for the given drive value
